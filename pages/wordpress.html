Introduction of WordPress

WordPress is the simplest and most popular way to create a website. It is a free Content Management System (CMS). CMS is a system which helps to manage content of your website. A website has different types of content like text, image, audio, video. WordPress is a very powerful and popular CMS system from other CMS systems in the world.


For create website with WordPress must need fulfill three requirement. Minimum requirement is

  1. Language- PHP
  2. Database- MySQL or MaraiDB
  3. Web server- Apache


Nowadays you will see more than one in four websites are WordPress websites. Wordpress powers over 33.6% of the top 10 million websites on the internet. So we can tell that WordPress is the most popular and powerful Content Management System solution in use.


Though Wordpress was originally created as a blog publishing system. But now it has increased for other types of web content. Now WordPress website use for any website. With WordPress you can create

  • Blog
  • eCommerce Store
  • Portfolios
  • Resumes
  • Social Networks
  • Mailing lists and
  • Forums.


WordPress was released on 27th may 2003. Its founded by american developer Matt mullenweg and English developer Mike Little.


To function wordpress, need to be installed on a web server, either part of an internet hosting service like wordpress.com. Or you can run it in a Computer with the software package wordpress.org in order to serve as a network host in its own right.


You can use a local computer for single user testing and learning purposes.


You can use readymade themes and plugins in WordPress. Some are totally free and some are paid. You can use both paid and free themes and plugins. In the free version you do not get the full feature of the theme and plugin.


Every wordpress website require one theme. For functionality and extend the features of a website require some plugins. If users have knowledge about themes and plugins users can change the look and functionality of a WordPress website without alerting the core code or site content.
